# Scolcoin-Smart-Chain
Blockchain Scol - Smart Contract

## Pre-requisitos
-  Go Ultimo Version
- Node

### Instalar Go
```
rm -rf /usr/local/go && tar -C /usr/local -xzf go1.19.linux-amd64.tar.gz
export PATH=$PATH:/usr/local/go/bin
go version
```
### Instalar Node
```
sudo apt update
sudo apt install nodejs
nodejs -v
```
